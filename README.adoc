= AsciidocEditor

This is a simple AsciiDoc editor and previewer

== Prerequisite
You will need https://jdk.java.net/12/[OpenJDK 12] and https://www.graphviz.org/[Graphviz].

== Building and Running
....
./gradlew build
....

....
./gradlew run
....

== Used Technologies
* https://openjfx.io/[JavaFX]
* https://ace.c9.io/[Ace Editor]
